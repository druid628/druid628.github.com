Here are some of the projects that I'm working on that I want to share with the world. Some projects are written for my company Blue Shamrock Development, some are written just for fun, while others are written to be shared. The following projects which are available or will be available on gitHub:
  • OddsThomas [php/symfony]- An online (faux) betting system. Currently incomplete mainly needs access to a feed which supplies odds (as well as a few other little pieces)
  • Mapstog [java] - A desktop app (swing) used to harvest email addresses from a web page. Used to explain to end users one of the many ways spammers harvest email addresses, when trying to get changes made to a website.
  • HP Hack [mixed] - (A collection of code that I did not write I wanted to keep available) Used for hacking the display of HP Printers.
  • cgJsonAdaptor[java] - a customizable package used in conjunction with gson to simplify encoding/decoding json. Package was built while working on an android app to simplify the amount of code to write. (promoting lazyness 1 line at a time)
